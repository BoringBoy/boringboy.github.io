两种方式安装Docker

# docker 安装

# 在线安装

  • centos7
  1. 准备工作
yum update

yum install -y yum-utils device-mapper-persistent-data lvm2

yum makecache fast
1
2
3

  1. 添加软件源信息
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
1
  1. 开始安装
yum -y install docker-ce
1
  1. 查询服务状态
systemctl status docker
1

# 离线安装

要离线安装docker，需要额外准备一台能联网的主机。其基本流程即是先联网将需要的软件包下载后，转移到没有连接网络的机器上安装。

  • Centos7
  1. 添加软件源信息
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
1
  1. 更新yum缓存:
yum makecache fast
1
  1. 创建下载目录并下载
#创建目录
mkdir -p  /home/dockerrpm/

#使用yum下载(只下载不安装)
yum intsall docker-ce --downloadonly --downloaddir=/home/dockerrpm/
1
2
3
4
5
6
  1. 打包rpm文件
tar zcvf docker-ce-20.tar.gz ./dockerrpm/*
1
  1. 目标机器上安装
#解压
tar zxvf docker-ce-20.tar.gz 

cd dockerrpm 
# 安装
rpm -ivh *.rpm --force --nodeps

  1. 查看docker状态

# docker 镜像源配置

# 创建或修改 /etc/docker/daemon.json 文件，修改为如下形式
{
    "registry-mirrors" : [
    "https://registry.docker-cn.com",
    "https://docker.mirrors.ustc.edu.cn",
    "http://hub-mirror.c.163.com",
    "https://cr.console.aliyun.com/"
  ]
}
# 重启docker服务使配置生效
$ systemctl restart docker.service
