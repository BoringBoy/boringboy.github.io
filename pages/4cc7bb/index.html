# 安装kubectl

# 方法1

可以从kubernetes库上直接下载，方法如下：

  1. 访问官方github网址：https://github.com/kubernetes/kubernetes/releases

  2. 找到想使用的发布版本，在每个发布版本的最后一行有类似“CHANGELOG-1.10.md”这样的内容，点击超链进入；

  3. 然后进入“Client Binaries”区域；

  4. 选择和目标机器系统匹配的二进制包下载；

  5. 解压缩，放入/usr/local/bin目录；

# 方法2

  1. 手动添加k8s仓库
cat <<EOF > /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=http://mirrors.aliyun.com/kubernetes/yum/repos/kubernetes-el7-x86_64
enabled=1
gpgcheck=0
repo_gpgcheck=0
gpgkey=http://mirrors.aliyun.com/kubernetes/yum/doc/yum-key.gpg
     http://mirrors.aliyun.com/kubernetes/yum/doc/rpm-package-key.gpg
EOF

  • 阿里云发布的minikube地址：https://github.com/AliyunContainerService/minikube
  1. 从release目录下载最新的minikube版本，然后执行：
chmod +x minikube
mv minikube /usr/local/bin
